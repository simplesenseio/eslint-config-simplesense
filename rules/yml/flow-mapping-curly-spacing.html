# yml/flow-mapping-curly-spacing

enforce consistent spacing inside braces

  • This rule is included in "plugin:yml/standard".
  • The --fix option on the command line (opens new window) can automatically fix some of the problems reported by this rule.

# Rule Details

This rule enforces consistent spacing inside braces of flow mappings.

# eslint yml/flow-mapping-curly-spacing: 'error'

#  GOOD
- {a: b}

#  BAD
- { a: b }

# Options

yml/flow-mapping-curly-spacing:
  - error
  - always # or "never"
  - arraysInObjects: false
    objectsInObjects: false

Same as object-curly-spacing (opens new window) rule option. See here (opens new window) for details.
