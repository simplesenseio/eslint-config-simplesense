# Prefer Date.now() to get the number of milliseconds since the Unix Epoch

This rule is enabled in the recommended config (opens new window).

This rule is automatically fixable by the --fix CLI option (opens new window).

Date.now() (opens new window) is shorter and nicer than new Date().getTime() (opens new window), and avoids unnecessary instantiation of Date objects.

# Fail

const foo = new Date().getTime();

const foo = new Date().valueOf();

const foo = +new Date;

const foo = Number(new Date());

const foo = new Date() * 2;

# Pass

const foo = Date.now();

const foo = Date.now() * 2;
