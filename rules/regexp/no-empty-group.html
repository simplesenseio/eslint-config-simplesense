# regexp/no-empty-group

disallow empty group

  • This rule is included in "plugin:regexp/recommended".

# Rule Details

This rule reports empty groups.

/* eslint regexp/no-empty-group: "error" */

/*  GOOD */
var foo = /(a)/;
var foo = /(?:a)/;

/*  BAD */
// capturing group
var foo = /()/;
var foo = /(|)/;
// non-capturing group
var foo = /(?:)/;
var foo = /(?:|)/;

# Options

Nothing.

# Version

This rule was introduced in eslint-plugin-regexp v0.1.0
